## Features

- Upload image functionality
- Real-time waste classification
- Support for JPG, JPEG, and PNG formats
- Simple and intuitive user interface
- Deep learning powered classification using CNN

- Python 3.7+
- TensorFlow

## Model Information

The model uses a Convolutional Neural Network (CNN) trained on a dataset of labeled waste images to classify waste as:
- Organic Waste
- Recyclable Waste

The model is built using TensorFlow/Keras and takes in images resized to 96x96 pixels.
